### 排布（非规则 4×2 网格）

| 帧 | 行 | 说明 | 左上角 (x, y) |
|----|-----|------|----------------|
| 0–3 | 上排 | 4 帧等宽对齐 | 0, 688, 1376, 2064 × **0** |
| 4–6 | 下排 | 3 帧，相对上排 **水平错开半帧**（落在上一行间隙下方） | 344, 1032, 1720 × **768** |

下排 x = `344 + n × 688`（`344 = 688 / 2`）。

组件内帧表：`LOADING_FRAMES` in `src/components/LoadingSprite/index.tsx`  
若换图后错开量不是半帧，只需改该数组坐标。
