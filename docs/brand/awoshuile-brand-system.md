# 阿窝睡了品牌体系（v1）

## 1. 品牌核心

- 品牌名：阿窝睡了
- 产品核心：结构化外化 + 微行动
- 核心目标：帮助用户在情绪混乱后恢复秩序，获得微小确定感
- 品牌角色：不催促、不评判的睡前陪伴者

## 2. 品牌叙事（产品环节映射）

| 产品环节 | 品牌叙事 |
| --- | --- |
| 用户打开 App | 进阿窝了：一个只属于你的安全角落 |
| 情绪入口（色块/身体感受） | 脱掉今天的外衣：把情绪先卸下来 |
| AI 引导式拆解（选择题） | 阿窝帮你理一理：像有人轻声问你怎么了 |
| 镜像重述 + 今日心智卡片 | 把今天存进枕头里：完成外化，不再带进梦里 |
| 微行动输出 | 睡前最后一件事：一个极小、可执行的动作 |
| 存档 / 树洞 | 睡吧，阿窝记得：明天醒来还能再看 |

## 3. 品牌语气规范

### 3.1 语气原则

- 温柔：使用低刺激表达，避免命令式语气
- 慢速：句子短，节奏缓，尽量一屏只传达一个动作
- 低压：不强调“必须改善”，只强调“可以先做一点”
- 非评判：不用“你应该/你不该”，改为“我们先/你可以”

### 3.2 文案写作规则

- 多用邀请式表达：例如“要不要先放下一点点”
- 少用诊断式表达：避免“你这是焦虑/你有问题”
- 优先当下动作：强调“今晚能做什么”，不追求长期说教
- 完成态强调“够了”：让用户获得“今天到这里也很好”的许可感

### 3.3 禁用表达

- 禁用：振作起来、你想太多了、别矫情、必须坚持
- 禁用：强对比鸡汤（例如“别人都在努力你不能停”）
- 禁用：未经请求的心理评估结论

## 4. Slogan 方案

- 主口号：每天睡前，来阿窝躺五分钟。
- 备用 1：把乱糟糟的今天，交给阿窝。
- 备用 2：阿窝睡了，你也睡吧。

## 5. 视觉方向（产品与品牌统一）

- 主色调：暖灰 + 柔和橘黄（模拟床头灯光）
- 图形母题：圆形“窝”，半闭眼状态，传达半睡半醒的安全感
- 动效基调：
  - 进入：像掀开一床软被
  - 退出：像灯光慢慢调暗
