# 阿窝睡了产品内文案包（v1）

## 1. 用户路径文案（六个核心环节）

## 1) 进阿窝了（打开 App）

- 主标题：阿窝睡了
- 辅助说明：今晚先不用把自己撑住，先在阿窝躺一会儿。
- 主按钮：进阿窝
- 次按钮：看看今天
- 空状态：你已经到了，剩下的慢慢来。

## 2) 脱掉今天的外衣（情绪入口）

- 主标题：先把今天放下来
- 辅助说明：选一个最贴近此刻的感受，不用解释得很完整。
- 主按钮：就这个感受
- 次按钮：我再看看
- 空状态：没有特别准确的词也没关系，先选一个最接近的。

## 3) 阿窝帮你理一理（AI 选择题拆解）

- 主标题：阿窝帮你理一理
- 辅助说明：我会轻轻问几个小问题，把脑子里的乱线慢慢分开。
- 主按钮：继续
- 次按钮：跳过这题
- 异常状态：如果这题不想答，我们先换一个角度也可以。

## 4) 把今天存进枕头里（镜像重述 + 心智卡片）

- 主标题：今天已经被好好接住了
- 辅助说明：这是你的今日心智卡片，先放进枕头里，今晚不用再反复想。
- 主按钮：收进枕头
- 次按钮：我想改一句
- 空状态：还没有生成卡片，我们再理一小步就好。

## 5) 睡前最后一件事（微行动）

- 主标题：睡前最后一件事
- 辅助说明：只做一个 2 分钟内能完成的小动作，今晚就够了。
- 主按钮：就做这个
- 次按钮：换一个更小的
- 完成提示：很好，今晚到这里就可以了。

## 6) 睡吧，阿窝记得（存档 / 树洞）

- 主标题：阿窝记得
- 辅助说明：你放下来的内容都在这里，明天醒来还能继续。
- 主按钮：去看看存档
- 次按钮：回到首页
- 空状态：这里还很安静，等你放下第一条今天。

## 2. 重情绪场景（极简引导）

- 文案 A：我在。你先选一个最接近的感受就好。
- 文案 B：先不用讲清楚，我们从一个小问题开始。
- 文案 C：今晚不求想通，只求放下一点点。

## 3. 完成态确认语（睡前最后一步）

- 版本 A：今天先到这里，已经很好了。
- 版本 B：你做完了睡前最后一件事，可以安心去睡了。
- 版本 C：把今天放进阿窝，明天再继续也来得及。

## 4. 匿名晚安卡片文案边界

- 入口说明：你可以匿名分享“今日心智卡片”，不会展示个人信息。
- 互动按钮：陪你睡
- 互动反馈：有人轻轻陪了你一下。

### 禁止项（保持安全感）

- 不开放评论
- 不开放私信
- 不展示用户主页跳转
- 不提供情绪打分排行
